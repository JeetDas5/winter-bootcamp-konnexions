# Winter BootCamp | Full Stack Web Dev | Konnexions

Hands-on 7-day bootcamp to go from the web fundamentals to building full-stack apps with React, Node.js, Express, MongoDB, and Next.js.

## Agenda at a Glance
- **Day 1:** HTML & CSS basics, layout, responsive design
- **Day 2:** JavaScript essentials, DOM manipulation, fetch API
- **Day 3:** React basics — components, props, state
- **Day 4:** React advanced — hooks, routing, state management patterns
- **Day 5:** Node.js, Express, MongoDB — building REST APIs
- **Day 6:** Advanced backend — auth, security, deployment tips
- **Day 7:** Next.js — routing, data fetching, API routes, deployment

## What You’ll Need
- A laptop with a modern browser (Chrome/Edge/Firefox)
- **Node.js 18+** and **npm** installed for JS/React/Node exercises
- A code editor (VS Code recommended)

## Quick Start
```bash
# clone the repo
git clone https://github.com/JeetDas5/winter-bootcamp-konnexions.git
cd winter-bootcamp-konnexions

# install dependencies for any provided samples
npm install

# run a sample frontend (if provided)
npm run dev

# run a sample backend (if provided)
npm run server
```

## Daily Focus
| Day | Topics | Outcomes |
| --- | ------ | -------- |
| 1 | HTML, CSS | Build a responsive landing page |
| 2 | JavaScript | Manipulate the DOM, handle events, call APIs |
| 3 | React Basics | Compose components, manage state/props |
| 4 | React Advanced | Use hooks, routing, and shared state patterns |
| 5 | Node, Express, MongoDB | Create REST endpoints and persist data |
| 6 | Advanced Backend | Add auth, validation, logging, and deployment prep |
| 7 | Next.js | Build SSR/SSG pages, API routes, and deploy |

## Suggested Workspace Layout
This codebase is organized into a folder per day; use the matching `day-0X/` directory for each set of exercises and examples.
- `Day_1_HTML_CSS/` HTML & CSS exercises
- `Day_2_JavaScript/` JavaScript exercises
- `Day_3_React_Basics/` React basics labs
- `Day_4_React_Advanced/` React advanced labs
- `Day_5_Backend_Basics/` Node/Express/MongoDB labs
- `Day_6_Backend_Advanced/` Advanced backend labs
- `Day_7_NextJS/` Next.js labs (routing, data fetching, API routes)

## How to Get the Most Out of This Camp
- Code along during sessions; keep iterating after.
- Commit small, frequent changes with clear messages.
- Ask questions early—conceptual clarity beats memorization.
- Pair up for debugging and reviews.
